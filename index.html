HiVLA: A Visual-Grounded-Centric Hierarchical Embodied Manipulation System
摘要总结

正文里它不是简单端到端 VLA，而是把高层语义规划和低层动作控制拆开：VLM 先做 task decomposition + visual grounding，输出子任务和目标框；再由 flow-matching DiT 动作专家结合全局图像、局部目标 crop 和语言技能信息生成动作。实验显示它在 RoboTwin 2.0 上比 H-RDT 和 π0 都强，尤其擅长长时程技能组合和小目标精细操作。

UMI-3D: Extending Universal Manipulation Interface from Vision-Limited to 3D Spatial Perception
摘要总结

这篇的核心是把 UMI 从纯视觉 SLAM 扩展成 wrist-mounted LiDAR + camera + IMU 的 3D 感知数据采集系统，用 LiDAR-centric SLAM 解决遮挡、动态场景和跟踪失败问题，生成更稳定的 metric-scale trajectory 和 3D 表示，再用同步后的 vision-action 数据训练下游策略。正文强调它不仅提升数据质量，还能支持原 UMI 很难做的 deformable object 和 articulated object 任务。

Goal2Skill: Long-Horizon Manipulation with Adaptive Planning and Reflection
摘要总结

正文里它是一个双系统长程操作框架：高层 VLM planner 维护结构化任务记忆，做目标分解、结果验证和失败后的反思式重规划；低层 VLA executor 在去干扰后的观测上做几何导向动作生成。实验在 RMBench 上把平均成功率做到 32.4%，显著高于最强基线 9.8%，尤其在 memory-intensive 子任务上提升更明显。

ESCAPE: Episodic Spatial Memory and Adaptive Execution Policy for Long-Horizon Mobile Manipulation
摘要总结

这篇主要解决长时程 mobile manipulation 的遗忘、空间不一致和执行僵化。它用 Spatio-Temporal Fusion Mapping 构建 depth-free 的持续 3D episodic memory，再用 Memory-Driven Target Grounding 做精确 interaction mask，最后通过 Adaptive Execution Policy 在全局导航和局部操作之间动态切换。正文给出的 ALFRED 结果很强：seen / unseen success rate 分别 65.09% / 60.79%，无详细指导时也能保持 61.24% / 56.04%。

Jump-Start Reinforcement Learning with Vision-Language-Action Regularization
摘要总结

正文的重点是把 VLA 当作“短期辅助引导”而不是永久 imitation：VLAJS 用 directional action-consistency loss 在训练早期给 RL 提供稀疏但高层的动作方向建议，从而改善探索和信用分配，同时不限制后期的最终性能。实验里它比 PPO 和 distillation 类方法更省样本，多个任务里可减少 50% 以上环境交互，且在 clutter、背景变化和外部扰动下更稳。

Beyond Conservative Automated Driving in Multi-Agent Scenarios via Coupled Model Predictive Control and Deep Reinforcement Learning
摘要总结

正文是把 MPC 当作结构先验，再让 DRL 在无信号交叉口的多车互动中学习更灵活的策略。它比纯 MPC 和纯 end-to-end RL 都更稳，在三个 traffic density 下都更好，collision rate 降了 21%，success rate 比纯 MPC 提升 6.5%。这类方法的优点是把安全性和适应性分层处理，比较接近可落地驾驶栈。

FeaXDrive: Feasibility-aware Trajectory-Centric Diffusion Planning for End-to-End Autonomous Driving
摘要总结

这篇的关键点是把扩散规划从“噪声中心”改成“轨迹中心”，直接围绕 clean trajectory 做可行性建模。正文里它同时做了三件事：训练时用自适应曲率约束压低局部几何不规则性，推理时用 drivable-area guidance 约束采样，后训练再用 feasibility-aware GRPO 进一步优化。它在 NAVSIM 上强调的是轨迹可行性和闭环规划表现同时提升，而不是只追单一 score。

DeCoNav: Dialog enhanced Long-Horizon Collaborative Vision-Language Navigation
摘要总结

正文里它解决的是双机器人长时程协同 VLN 的同步执行和在线重分工问题。方法上有三个核心：Semantic Visual Bus 共享紧凑语义状态，Event-driven Dialogue Replanning 根据新证据重分配子任务，Synchronous Parallel Execution 保证共享时钟下的并行推理。DeCoNavBench 由 1,213 个任务、176 个 HM3D 场景组成，整体 both-success rate 提升 69.2%，更适合 relay/handoff 这类协作任务。

Unveiling the Surprising Efficacy of Navigation Understanding in End-to-End Autonomous Driving
摘要总结

正文指出很多 end-to-end driving 系统其实过度依赖局部感知，几乎没真正利用 global navigation info。于是作者提出 Sequential Navigation Guidance：把全局路线拆成 navigation path + turn-by-turn 指令，并构建 SNG-QA 让全局导航和局部规划对齐。SNG-VLA 在正文里强调的是：不靠额外 perception loss，也能更准确地把导航信息吃进去，适合把真实地图/API 导航信息接进驾驶系统。

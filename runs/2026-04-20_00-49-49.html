DockAnywhere: Data-Efficient Visuomotor Policy Learning for Mobile Manipulation via Novel Demonstration Generation
2604.15023

这篇主要解决移动操作里“换一个 docking 点就掉点”的视角泛化问题。核心做法是把**单条示范**扩展成多个可行 docking 位置上的新示范：先把 base motion 和接触操作解耦，再用点云级的空间编辑合成不同视角下的观测与动作。论文在 ManiSkill 和真实平台上都报告了明显提升。

实现概率**83%**
开源**yes** — abstract 里直接给了 GitHub URL。
可部署**yes** — 明确做了真实世界 mobile manipulation 验证，目标就是提升部署泛化。

World-Value-Action Model: Implicit Planning for Vision-Language-Action Systems
2604.14732

这是一个面向 VLA 的**隐式规划**框架，不再直接做 action prediction，而是把未来轨迹压到 latent space 里：一个 world model 预测未来状态，一个 trajectory value function 评价长期价值，动作生成则变成在 latent 空间里做推断。论文强调它比直接在 action space 里规划更适合长时域决策，并在仿真和真实实验里都优于基线。

Goal2Skill: Long-Horizon Manipulation with Adaptive Planning and Reflection
